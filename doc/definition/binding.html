Binding

A Binding object represents a class or namespace.

Attributes

  • source: (required) the C++ class/namespace name.
  • target: (required) the target language class/namespace/package/module name; this is specified in the format Parent::Child. Target languages may need to adjust this name; for example, Python changes it to Parent.Child.
  • version: (required) the version number for this binding.
  • source-inherits: a space-separated list of C++ base class names.
  • target-inherits: a space-separated list of target language base class names.
  • must-not-delete
  • doc: a documentation string.

Children

  • properties: object properties to expose.
  • functions: class/object methods or namespace functions to expose.
  • operators: overridden operators to implement.
  • constants: constants to expose.
  • globals: globals to expose.

Properties

A Properties object represents object properties to expose.

Attributes

none

Children

  • property: (zero or more) properties.

Property

A Property object represents an object property to expose.

Attributes

  • name: (required) the property name.
  • doc: a documentation string.
  • type
  • string-length
  • max-string-length
  • array-length
  • max-array-length
  • pass-as-pointer

Children

none

Operators

An Operators object represents overridden operators to expose.

Note: If they are not defined here, the == and != operators are automatically defined by the generator to compare the pointers of the wrapped C++ objects. This allows different target language objects that wrap the same C++ object to be treated as equal.

Attributes

none

Children

  • operator: (zero or more) operators.

Operator

An Operator object represents an overridden operator.

The following operators are currently supported:

  • neg (unary -)
  • ==
  • !=
  • +
  • - (binary -)
  • &
  • |
  • +=
  • -=

Attributes

  • name: (required) the operator name.
  • doc: a documentation string.

Children

none

Constants

A Constants object represents constants to expose.

Attributes

none

Children

  • constant: (zero or more) constants.

Constant

A Constant object represents a constant to expose.

Attributes

  • name: (required) the constant name.
  • group: a space-separated list of groups the constant belongs to; Perl, for example, uses this to export constants to the calling namespace in groups rather than singly or all at once.
  • doc: a documentation string.
  • type
  • string-length
  • max-string-length
  • array-length
  • max-array-length

Children

none
